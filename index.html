Intelligence Artificielle

L'Intelligence Artificielle est la capacité d'une machine à miner le comportement humain

Branches de l'IA

  • Le machine learning
  • Le Deep learning
  • Le Natural Language Processing(NLP)

Le Machine Learning

Inventé par Paul Samuelson en 1959, le machine learning est l'apprentissage statistique automatique. Il consiste à laisser la machine apprendre quel calcul effectuer pour resoudre un problème plutôt que de le programmer de façaon explicite.

La petite question qu'on pourrait se poser est: Comment apprendre à une machine ? autrement dit, quelles sont les méthodes d'apprentissage d'une machine?

    • L'apprentissage supervisé
    • L'apprentissage non supervisé
    • L'apprentissage par renforcement

L'apprentissage supervisé

Type d'apprentissage dans lequel la cible ou target est connue
Apprentissage dans laquelle la machine apprend par exemple à partir des correlations entre les différentes variables et la cible
Quelques algorithmes utilisés sont: la regression logistique, les support vecteur machine...

L'apprentissage non supervisé

Type d'apprentissage dans lequel la cible ou target est inconnue
Apprentissage dans laquelle la machine apprend toute seule les correlations entre les différentes variables ou caractéristiques du problème
L'un algorithme utilisés est: Les K-Means généralement utilisés pour la segmentation des Clustering...

L'apprentissage par renforcement

Type d'apprentissage dans lequel on corrige la machine en fonction de l'objectif à chaque fois qu'elle se trompe.clinez pour aller loin
Apprentissage dans laquelle la machine apprend toute seule les correlations entre les différentes variables ou caractéristiques du problème
